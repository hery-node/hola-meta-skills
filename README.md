# Hola Meta Skills

AI coding skills for the **Hola meta-programming framework**. Each skill teaches an AI assistant how to work with specific patterns in `hola-server` and `hola-web`.

## Skill Index

### Core Framework

| Skill | Description |
|-------|-------------|
| [hola-meta](skills/hola-meta/SKILL.md) | Entity metadata definition with `init_router()` — field visibility flags, protected fields, field subsets, role-based access |
| [hola-router](skills/hola-router/SKILL.md) | RESTful API endpoints auto-generated by `init_router()` — all routes, params, responses, and permissions |
| [hola-hooks](skills/hola-hooks/SKILL.md) | Lifecycle hooks (`before_create`, `after_update`, `list_query`, etc.) for validation, defaults, and business logic |
| [hola-add-meta-type](skills/hola-add-meta-type/SKILL.md) | Custom type definitions for field validation — server/client registration, 35+ built-in types, helper functions |
| [hola-web-code](skills/hola-web-code/SKILL.md) | Standardized HTTP status and error codes for server responses |
| [hola-i18n](skills/hola-i18n/SKILL.md) | Internationalization — translating labels, adding languages, localizing field names |

## Usage

Copy the `skills/` directory into your project's `.agent/skills/` folder:

```bash
cp -r skills/ /path/to/your-project/.agent/skills/
```

The AI assistant will automatically discover and use these skills when relevant tasks are requested.
